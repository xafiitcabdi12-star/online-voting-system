# Online Voting System

## Project Description
The Online Voting System is a web-based application developed using PHP and MySQL.
The system is designed to allow users to vote online in a secure and organized way.
It helps manage voters, candidates, and voting results efficiently.
This project is developed for academic purposes.

## Database Setup Instructions

2. Create a new database (e.g., online_voting_system)
3. Import the provided .sql file
4. Configure database connection in the PHP files if needed

---

## How to Run the Project

1. Install XAMPP
2. Copy the project folder into the htdocs directory
3. Start Apache and MySQL
4. Open browser and go to:
   http://localhost/online_voting_system
